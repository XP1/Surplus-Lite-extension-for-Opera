Surplus Lite

Made by antimatter15.

Ported by XP1.

Lightweight Google+ notifications in Opera.

Based on @antimatter15's Google Chrome extension.

User

Enter your Google+ user number if you are using the multi-login feature with other Google+ profiles. By default, if you only have one Google+ profile, your user number is "0".

Updater

The updater checks for new, unread notifications and updates the toolbar button with the number of notifications.

If enabled, the updater will check continously for new, unread notifications even if it exceeds the maximum number of retries. If disabled, the updater will obey and give up after it reaches the maximum number of retries, and the updater will also disable itself.

The maximum number of retries will apply when retry continously is disabled. After the updater has reached the maximum number of retries consecutively, the updater will disable itself and no longer check for new, unread notifications.

In milliseconds, the count update request interval is the time between update requests. By default, the interval is 15 seconds.

Powered by the Opera browser.
